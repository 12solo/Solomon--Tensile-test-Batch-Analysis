Solomon Tensile Suite 2 🚀Analytical Framework for Bio-Composite Strain BehaviorThe Solomon Tensile Suite 2 is a high-fidelity mechanical characterization tool built for Material Scientists and Engineers. While optimized for biodegradable polymers—specifically PBAT and PBAT/PLA blends—it provides a robust solution for batch-processing raw tensile data, providing automatic peak detection, and elastic region validation.✨ Key FeaturesMulti-Source Ingestion: Supports standard Excel (.xlsx), CSV, and TXT (instrument raw data) files.Integrated Plot Digitizer: Extract data directly from legacy plot images using an interactive canvas.Automatic Peak Detection: Curves are automatically limited to the point of Maximum Stress (Ultimate Tensile Strength) to ensure accurate elongation-at-break calculations.Interactive Modulus Validation: Real-time "Modulus Fit" sliders with visual feedback to ensure the elastic slope matches instrument standards (e.g., matching the 150.6 MPa baseline).Toe-Region Compensation: Mathematical shift to $(0,0)$ to remove initial slack from the mechanical grips.Batch Analytics: Process dozens of samples simultaneously with Mean, Standard Deviation, and Specimen Count ($n$) reporting.🛠️ Technical FrameworkThe suite calculates constitutive parameters based on standard engineering mechanics:Engineering Stress ($\sigma$): Calculated as $\frac{F}{A_0}$. The app automatically detects if your file contains an instrument-calculated "Sforzo" (Stress) column to prioritize accuracy.Engineering Strain ($\epsilon$): Derived from $\frac{\Delta L}{L_0} \times 100$, where $L_0$ is the user-defined initial gauge length.Young’s Modulus ($E$): Determined via linear regression ($\sigma = E\epsilon + b$) within the user-adjustable elastic range.Work Done & Toughness: Calculated using trapezoidal integration of the area under the stress-strain curve.
